## Vorbereitung

### Was macht das Skript?
Das Skript richtet die komplette Lab-Umgebung ein:

Gruppen anlegen: music, wrestle, emperors
User anlegen und Gruppen zuweisen:

elvis → Gruppe music
blondie → Gruppen music, wrestle
hogan → Gruppe wrestle
nero → Gruppe emperors

Passwörter aller User auf kali setzen
Verzeichnisstruktur /tmp/lab/gigs/, /tmp/lab/drafts/ erstellen
Besitzer auf root:root setzen, Rechte rekursiv auf 755

Ausführung:
``sudo bash setup.sh``
